Модуль «Оборудование»

  • К элементам оборудования для разливного пива относятся:

    Охладитель
    Предназначен для охлаждения пива до оптимальной температуры.
    Разливная колонна
    Несет рекламную и функциональную нагрузку. Внутри нее размещен пивопровод и канал водяного доохлаждения пивопровода.
    Кег (пивная бочка)
    Баллон с газом (СО2, для некоторых сортов пива дополнительно используют азот).
    Редуктор
    Отвечает за поддержание необходимого давления углекислого газа для движения пива по системе.
    Бочковая головка (Коннектор)
    Обеспечивает герметичное соединение кега с пивопроводом.(Открывает клапана подачи пива и газа).
    Адаптер
    Служит для промывки пивопровода, позволяет произвести подсоединение пивопровода к проточной воде.
